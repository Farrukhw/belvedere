Folders Tab

The Folders Tab is the main working area of the application. Here the users assign rules to the specific folders that they have selected.

The tab is divided into two main sections, the left half where the folders are selected, and the right, where the rules for the selected folder are displayed.

Folder List

The list shown here are the folders that have been added to Belvedere to monitor with the rules displayed on the right side.

The following buttons are available below the folder list:

  • '+' Button - To add a folder, click the '+' button at the bottom of the screen, and select the folder from the dialog box.
  • '-' Button - To remove a folder, click the '-', and confirm your deletion of the folder. If you delete a folder with rules, the rules will be removed as well.

To display the rules assigned to a particular folder, left click to highlight the folder, and the assigned rules will be displayed in the right side of the application.

Rule List

The list shown here are the rules for the folder selected in the left side of the screen. Here you can see the name of the rule, and if the rule is currently enabled or not. If a rule is listed as 'Yes' in the 'Enabled' column, the rules will be applied to all files within the specified folder. If the rule is listed as 'No' in the 'Enabled' column, this particular rule will be skipped over during processing.

The following buttons are available below the folder list:

  • '+' Button - To add a rule, click the '+' button at the bottom of the screen and complete the Create a rule... dialog.
  • '-' Button - To remove a rule, click the '-', and confirm your deletion of the rule.
  • Edit Rule Button - To edit an existing rule, highlight the rule above, and click the 'Edit Rule' button and complete the Edit Rule Dialog.
