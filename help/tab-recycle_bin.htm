Recycle Bin Tab

With the Recycle Bin tab, users can configure the Recycle Bin behaviors available in Belvedere



Right now we have two sections within this screen:

Enable Belvedere Recycle Bin Management
Empty Recycle Bin Intervals

Allow Belvedere to Manage...

This checkbox enables the rest of the tab contents for editing. We use this as an additional safeguard to protect anything from being turned on by accident. Once this is checked the rest of the screen will be available for editing.

Top

Empty Recycle Bin Intervals

This setting tells Belvedere how often to empty your recycle bin. It can be set in minutes, hours, days, and weeks, if you happen to let Belvedere run for periods that long. The timer starts from the first time it is enabled, or when Beleveder starts if it was enabled when Belevdere was started.
