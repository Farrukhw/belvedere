Tray Menu

The tray menu is available by right clicking on the icon in your system tray. The tray menu has the following options available to the users:



Manage
Preferences
Pause
About...
Exit

Manage Menu Option

The manage menu option opens the main Belevedere program to allow for normal system use.

Top

Preferences Menu Option

The preferences menu option opens the preferences tab, where users can configure the various aspects of Belvedere to their liking.

Top

Pause Menu Option

The pause menu option allows users to pause the application so that no folders are rules are processed until the user 'un-pauses' the application. A check mark next to the pause menu option in the File Menu, Tray Menu, and a RED tray icon denotes that Belvedere is in a paused state.

Top

About Menu Option

The about belvedere menu option allows user to see the application version and other pertinent information about the program.

Top

Exit Menu Option

The exit menu option allows user to completely end the application from executing.
